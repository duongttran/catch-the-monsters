Catch the Monsters!

Rules of the game:

  1. I have 15 seconds to catch as many monsters as I can.
  2. If I catch a monster then another monster is randomly placed on the screen.
  3. I see my score updated when I catch a monster.
  4. If the timer runs out I cannot move my hero anymore.
  5. If the timer runs out I can press the reset button and start the game over.
